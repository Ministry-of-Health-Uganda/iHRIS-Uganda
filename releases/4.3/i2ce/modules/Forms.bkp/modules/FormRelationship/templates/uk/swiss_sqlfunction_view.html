SQL функція --

SQL-запит:

Визначальний SQL для цієї функції. Ви звертаєтеся до оголошених полів як 'reportform+field'

Поле форми:

Поле форми, в якому ця функція приймає значення

Форми для вибору

Форми, вибрані для цього поля. Застосовується тільки, якщо поле форми є підкласом I2CE_FormField_MAPPED. У цьому випадку поле повинно бути розділеним комами списком форм.

Зв'язані поля

Поля, які пов'язані при перерахуванні варіантів. Застосовується тільки, якщо поле форми є підкласом I2CE_FormField_MAPPED. Воно повинно бути списком відокремним стовпцем виду "форми" чи "форма+поле посилання". Приклад: "повіт:район+регіон:[регіон]:країна".
