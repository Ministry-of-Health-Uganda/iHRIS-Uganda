Formulários selecionáveis

Os formulários que são selecionáveis para este campo. Aplica-se apenas se o campo do formulário for uma sub-classe de I2CE_FormField_MAPPED. Em cujo caso deveria ser uma lista de formulários separados por vírgula. Se não estiver definido, será o nome desta função, por padrão.

Campos linkados

Os campos que estão ligados quando se listam as opções. Aplica-se apenas se o campo do formulário for uma sub-classe de I2CE_FormField_MAPPED. Deveria ser uma lista separada por "dois pontos" (:) de 'forms' ou 'form+linkfield's. Exemplo: "county:district+region:[region]:country". Se não estiver definido, será o nome desta função, por padrão.
