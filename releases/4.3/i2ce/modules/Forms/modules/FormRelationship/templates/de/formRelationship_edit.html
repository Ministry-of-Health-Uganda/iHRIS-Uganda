Du kannst entweder eine vorhandene Datenblattbeziehung anpassen oder eine neue erstellen. Du kannst eine vorhandene Datenblattbeziehung auf dieser Seite aufbauen.

Eine vorhandene Datenblattbeziehung ansehen

    Eine neue Datenblattbeziehung erstellen

    Bitte entweder ein Datenblatt als primäres Datenblatt für diese Datenblattbeziehung auswählen oder kopiere eine vorhandene Datenblattbeziehung in die neue Datenblattbeziehung, indem Du unten auf den entsprechenden Link bei den Optionen klickst. Bitte ein Datenblatt als primäres Datenblatt für diese Datenblattbeziehung auswählen indem Du oben auf den Link zu den Optionen klickst.

    Datenblattbeziehungskurzname:

    Ein interner Schlüssel der zur Identifizierung der Datenblattbeziehung genutzt wird. Er darf nicht einem der Schlüssel für vorhandene Datenblätter dieser Datenblattbeziehung entsprechen.

    Anzeigename:

    Der Name der Datenblattbeziehung, der dem Anwender angezeigt wird.

    Beschreibung:

    Eine Beschreibung der Datenblattbeziehung, die dem Anwender verständlich sein sollte.
    Um eine neue Datenblattbeziehung aufzubauen, muss zunächst ein primäres Datenblatt ausgewählt werden.
    Eine vorhandene Datenblattbeziehung zum Kopieren auswählen.
