Редагувати SQL функцію

SQL-запит:

Визначальний SQL для цієї функції. Ви звертаєтеся до оголошених полів як 'reportform+field'

Опис:

Опис цієї функції, який відображається для кінцевих користувачів.

Поле форми:

Виберіть поле форми, в якому ця функція приймає значення

З'єднати:

Розділений комами список форм у зв'язках, на полях якого ви хочете накопичувати (тобто вони не є згуртованими, вони є формами, які з'являються у вашій функції)

Форми для вибору

Форми, вибрані для цього поля. Застосовується тільки, якщо поле форми є підкласом I2CE_FormField_MAPPED. У цьому випадку поле повинно бути розділеним комами списком форм. Якщо не вказано, то типово воно збігається з ім'ям цієї функції.

Зв'язані поля

Поля, які пов'язані при перерахуванні варіантів. Застосовується тільки, якщо поле форми є підкласом I2CE_FormField_MAPPED. Воно повинно бути списком відокремним стовпцем виду "форми" чи "форма+поле посилання". Приклад: "повіт:район+регіон:[регіон]:країна". Якщо не вказано, то за замовчуванням воно збігається з ім'ям цієї функції.
