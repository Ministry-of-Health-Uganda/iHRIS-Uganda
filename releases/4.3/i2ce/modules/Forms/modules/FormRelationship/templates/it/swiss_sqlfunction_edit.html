Modifica la funzione SQL

Sql Query:

La definizione di SQL per questa funzione. Si fa riferimento ai campi segnalati come 'reportform + campo'

Descrizione:

Una descrizione di questa funzione che verraà visualizzata dall'utente finale

Campo del modulo:

Seleziona il campo del modulo da cui questa funzione prende il valore

Aggrega:

Un elenco separato da virgole dei moduli in relazione a quei campi che vuoi aggregare (es. quelli che non sono nella clausola groupby, quelli sono i moduli che appaiono nella tua funzione)

Moduli selezionabili

I moduli selezionabili per questo campo. Si applica solo se il campo del modulo è una sottoclasse di I2CE_FormField_MAPPED. In tal caso dovrebbe essere un elenco separato da virgole di moduli. Se non indicato, il valore predefinito sarà il nome di questa funzione.

Campi collegati

The fields that are linked when listing options. Only applies if the form field is a subclass of I2CE_FormField_MAPPED. It should be a colon separated list of 'forms' or 'form+linkfield's. Example: "county:district+region:[region]:country". If not set, it defaults to the name of this function.
