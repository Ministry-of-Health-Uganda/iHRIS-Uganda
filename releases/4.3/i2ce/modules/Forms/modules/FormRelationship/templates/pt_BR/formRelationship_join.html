Adicionar um novo formulário
Selecione um formulário filho para juntar com esse formulário:

Abreviação:

An internal key used to identify the form relationship. It may not be the same as the key for any of the existing named forms in this relationship.

Nome de exibição:

O nome do formulário resultante que é mostrado ao usuário final.

Descrição:

Uma descrição do formulário e seu papel na relação, que deve ser legível para o usuário final.
