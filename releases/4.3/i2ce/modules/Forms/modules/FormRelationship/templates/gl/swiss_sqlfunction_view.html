A función SQL --

Pescuda SQL:

A pescuda SQL para esta función. Cómpre referirse aos campos dos que se informou como `formulariodeinforme+campo`

Campo de formulario:

O campo de formulario desde o que a función toma os valores

Formularios seleccionables

Os formularios seleccionables para este campo. Unicamente se aplica se o campo do formulario é unha subclase de I2CE_FormField_MAPPED. Nese caso, debe ser unha lista de formularios separados por comas.

Campos ligados

Os campos ligados ao listar as opcións. Unicamente se aplica se o campo do formulario é unha subclase de I2CE_FormField_MAPPED. Debe ser unha lista separada por dous puntos de 'formularios' ou 'formulario+campoligado'. Exemplo: "país:distrito+rexión:[rexión]:país".
