La requête SQL de cette fonction. On se refère aux champs du rapport par la syntaxe : `Rapport+Champ`

Description:

Une description de cette fonction à afficher pour l’utilisateur final

Champ de formulaire:

Sélectionner le champ de formulaire dans lequel cette fonction prend des valeurs

Agréger:

Une liste séparée par des virgules des formulaires en lien avec ces champs que vous voulez agréger (c.à.d. ceux qui ne sont pas dans la clause groupby, ce sont les formulaires qui apparaîtront dans votre fonction)

Formulaires sélectionnables

Les formulaires qui peuvent être sélectionnés pour ce champ. S'applique uniquement si le champ du formulaire est une sous-classe de 2CE_FormField_MAPPED. Dans ce cas, il doit être une liste de formulaires séparée par des virgules. S'il n'est pas positionné, sa valeur par défaut sera le nom de cette fonction.

Champs liés

Les champs qui sont liés quand on fait la liste des options. Appliquer uniquement si le champ de formulaire est de la sous-classe I2CE_FormField_MAPPED. Il devait être une liste ‘des formulaires’ ou ‘formulaires+champlié’ séparé par deux-points. Par exemple: «pays:district+région:[région]:pays». S’il n’est pas précisé, le nom de fonction par défaut sera utilisé.
