A função SQL --

Query Sql:

The defining SQL for this function. You refer to reported fields as `reportform+field`

Campo de Formulário:

O campo do formulário que essa função preenche

Formulários selecionáveis

Os formulários que são selecionáveis para este campo. Aplica-se apenas se o campo do formulário for uma sub-classe de I2CE_FormField_MAPPED. Em cujo caso deveria ser uma lista de formulários separados por vírgula.

Campos linkados

The fields that are linked when listing options. Only applies if the form field is a subclass of I2CE_FormField_MAPPED. It should be a colon separated list of 'forms' or 'form+linkfield's. Example: "county:district+region:[region]:country".
