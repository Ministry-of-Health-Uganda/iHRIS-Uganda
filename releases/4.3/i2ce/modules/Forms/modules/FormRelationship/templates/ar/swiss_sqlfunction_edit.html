تعديل وظيفة SQL

SQL الاستعلام:

ال SQL المعرف لهذه الوظيفة. قم بالرجوع الي الحقول ال مقررة مثل تقرير النموذج + حقل`

توصيف:

توصيف المهمة ليتم عرضها للمستخدم النهائي

حقل النموذج:

حدد حقل النموذج المستخدم بواسطة المهمة لتعبئة القيم

إجمالي:

A comma separated list of the forms in the relationship on whose fields you which you wish to aggregate on (i.e. those not in the groupby clause, these are the forms that appear in your function)

النماذج التي تم اختيارها

The forms which are selectable for this field. Only applies if the form field is a subclass of I2CE_FormField_MAPPED. In which case it should be a comma separated list of forms. If not set, it defaults to the name of this function.

الحقول المرتبطة

الحقول المرتبطة بقائمة الخيارات. لا ينطبق إلا إذا كان حقل النموذج هو فئة فرعية من I2CE_FormField_MAPPED. فهي يجب ان تكون قائمة جدول منفصل من "النماذج" أو "شكل + في linkfield. على سبيل المثال: "مقاطعة: المنطقة + المنطقة: [المنطقة]: البلد". إذا لم يتم ضبطها سوف تعود الي الإسم المفترض للخاصية.
