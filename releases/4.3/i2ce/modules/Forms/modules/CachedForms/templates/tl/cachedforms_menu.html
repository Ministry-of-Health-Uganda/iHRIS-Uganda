Itago ang mga Pormularyo

Itago ang lahat ng mga pormularyo sa loob ng napiling balangkas.
Pumili ng isang balangkas:

Taguan lahat ng mga pormularyo
Sapiling Pagtatago ng lahat ng mga pormularyo kung hindi bilasa
Ibagsak mga taguan ng lahat ng mga pormularyo

Iluwas ang Itinagong mga Pormularyo

Iluwas ang lahat ng mga pormularyong nakatago na nasa loob ng napiling balangkas. Kapag pumili ka ng isang petsa ng pagbabago, kung gayon ang mga rekord lamang na nabago noon o pagkalipas ng tinukoy na petsa ang iluluwas. Kung hindi, iluluwas ang lahat ng mga rekord na nabago para sa napiling balangkas.
