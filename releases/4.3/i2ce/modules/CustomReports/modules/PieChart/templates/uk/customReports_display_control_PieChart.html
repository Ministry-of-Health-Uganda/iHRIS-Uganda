Виберіть стиль діаграми.
Вибрати спосіб, яким ви хочете вибрати поля для відображення.

Одне поле - всього

Одне поле (всього) Відображає одне поле із загальною чисельністю, з цим значенням для цього поля.

Два поля - всього на другому полі

Відображає два поля із загальною чисельністю на другому полі. Два поля (всього на другому)

Два поля - друге є числовим

Відображає два поля, одне з яких має числове значення Два поля (числові)

Всього на два поля

Відображає всього на два поля Два поля (Всього)

Три поля - всього на третьому

Три поля (Всього на Третьому)


Три поля - третє є числовим

Відображає три поля, одне з яких є числовим. Три поля (числові)
