Ви можете вибрати або змінити наявний зв'язок форми, або побудувати новий. Може побудувати наявну форму зв'язків з цієї сторінки.

Редагувати наявний зв'язок

    Побудувати новий зв'язок

    Будь ласка, виберіть форму для використання в якості основної форми для цієї форми зв'язків або виберіть, щоб скопіювати наявну форму зв'язків на нову, натиснувши на відповідне посилання нижче Параметрів. Будь ласка, виберіть форму для використання в якості основної форми для цієї форми зв'язків, натиснувши на посилання нижче Параметрів.

    Коротке ім'я зв'язку:

    Внутрішній ключ використаний для ідентифікації форми зв'язків. Він не може бути таким же, як ключ для будь-яких наявних форм зв'язків.

    Показати ім'я:

    Назва форми зв'язку, яка відображається для кінцевих користувачів.

    Опис:

    Опис форми зв'язків, які повинні бути зрозумілі користувачеві.
    Щоб побудувати зв'язок нової форми, вам необхідно спочатку вибрати основну форму
    Виберіть наявний зв'язок для копіювання.
